# GREEN BROTHERS – Градинарски услуги

Професионален уебсайт за GREEN BROTHERS – бизнес за градинарски услуги, косене на дворове, подрязване на храсти, почистване на парцели и поддръжка на градини.
link: greenbrothers.netlify.app
## Услуги

* Косене на тревни площи
* Подрязване на храсти и жив плет
* Почистване на дворове и парцели
* Събиране и извозване на зелени отпадъци
* Абонаментна поддръжка на градини
